# MyCV - Personal Portfolio Webpage

Welcome to my personal CV repository! This project is a custom-built, responsive webpage that serves as my professional online CV and portfolio. It was designed to introduce my skills, projects, and personality to the world as I progress through my Software Engineering journey.

## ✨ Features
* **About Me:** A brief look into who I am, my interests (from Lego and dolls to video games and coding), and my goals.
* **Skills & Competencies:** A highlight of my technical stack, including Python and Django.
* **Project Showcase:** Summary of my work, including my News Application capstone and game logic projects.
* **Contact Info:** Easy links to contact me via email or phone number.
* **Responsive Design:** Styled with CSS to look clean on different screen sizes.

## 🛠️ Technologies Used
* **HTML5:** For the structure and content.
* **CSS3:** For the custom pastel styling, layout, and aesthetic.
* **GitHub Pages:** Used for hosting and deployment.
